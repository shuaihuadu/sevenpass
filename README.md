# 逢7过 · SevenPass

考验反应力与专注力的数字游戏，支持 iOS 与 Android。

🌐 **官网** → [shuaihuadu.github.io/sevenpass](https://shuaihuadu.github.io/sevenpass)

> 我得了 998 分，你敢来挑战吗？——「炫耀一下」

## 游戏规则

- 与 AI 轮流报数
- 7 的倍数或含 7，必须说「**过**」或「**拐**」
- 「**说**」报出当前数字
- 「**过**」跳过该数字
- 「**拐**」可反转报数方向
- 答错或超时即失败
- 彩蛋：遇到 0，你 +5 分；AI 遇到则 −5 分
