# 逢7过

微信小游戏 · 经典逢7过 · 无限关卡 · 四种难度

## 游戏规则

数字逐个出现，在限定时间内做出判断：

| 情况                | 操作     | 示例                       |
| ------------------- | -------- | -------------------------- |
| 数字**包含 7**      | 点「过」 | 7, 17, 27, 70, 71, 77...   |
| 数字**是 7 的倍数** | 点「过」 | 14, 21, 28, 35, 42, 49...  |
| 其他普通数字        | 点「说」 | 1, 2, 3, 4, 5, 6, 8, 9... |

> **疯狂模式额外规则**：各位数字之和含 7 或是 7 的倍数，也需点「过」  
> 例：16 → 1+6=7 → 「过」；29 → 2+9=11（非7相关）→ 「说」

- **只有 1 条命**，答错或超时立即结束
- 失败后显示该数字的判断原因
- 每种难度独立记录最高分（无限关卡）

## 难度说明

| 难度 | 数字范围 | 反应时限 | 特殊规则 |
| ---- | -------- | -------- | -------- |
| 简单 | 顺序数字（每关 20 个） | 约 2.2 秒 | 无 |
| 中等 | 随机两~四位数（每关 12 个） | 约 1.8 秒 | 无 |
| 高级 | 随机两~四位数（每关 15 个） | 约 1.2 秒 | 无 |
| 疯狂 | 随机两位数起无上限（每关 15 个） | 约 1.0 秒 | 含位数和规则 |

关卡越高，数字范围越大、时限越短。

## 技术栈

- 微信小游戏原生 Canvas 2D API
- ES6 模块化（import / export）
- 无第三方依赖，轻量纯原生

## 快速开始
